# 🛒 Retail Store Sales Analysis

## 📌 Project Overview

This project analyzes retail sales data of electronic products across different cities such as Pune, Mumbai, and Nagpur. The main objective is to extract meaningful insights like top-performing products, revenue distribution, and sales trends using Python, SQL, and Excel.

---

## 🎯 Objectives

* Calculate total sales revenue
* Identify top-selling products
* Analyze city-wise and category-wise performance
* Visualize sales trends over time
* Provide insights for better business decision-making

---

## 🧰 Tech Stack

* Python (Pandas, Matplotlib)
* SQL
* Microsoft Excel

## 📈 Key Insights

* Laptops generate the highest revenue among all products
* Pune contributes the highest overall sales
* Mobile sales perform consistently across cities
* Accessories like keyboards and headphones contribute less revenue
